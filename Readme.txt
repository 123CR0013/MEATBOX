■操作説明

キーボードでの操作

・上下左右キー
操作キャラクターの移動
・zキー
現在のステージを最初からリトライ
・qキー
次のステージへ
・wキー
ボスステージへ
・スペースキー、または、zキー
ページ送り（タイトル画面、説明画面、ゲームオーバー画面）


■概要説明


 「MEATBOX」は倉庫番というパズルゲームを参考にした、パズルとアクションを合体させたゲームです。
プレイヤーはステージ内の箱を押し、ステージに配置された敵を挟んで、倒します。
ステージにいるすべての敵を倒すとステージクリアです。
9ステージ+ボスステージの全10ステージです。
